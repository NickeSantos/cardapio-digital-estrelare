# 🍽️ Cardápio Digital Estrelare

Projeto frontend desenvolvido com HTML, CSS e JavaScript para simular o cardápio digital de um restaurante fictício.

## 🚀 Funcionalidades

- Navegação entre categorias;
- Exibição de produtos;
- Layout responsivo;
- Interface intuitiva;
- Organização por seções do cardápio.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📋 Categorias Disponíveis

- 🍔 Lanches
- 🍕 Pizzas
- 🥟 Pastéis
- 🥤 Bebidas
- 🍨 Sorvetes
- 🍰 Sobremesas
- 🍹 Drinks
- 🥙 Esfihas
- 🎁 Combos

## 🎯 Objetivo

Projeto acadêmico desenvolvido para praticar conceitos de desenvolvimento web, estruturação de páginas, estilização responsiva e interatividade utilizando JavaScript.

> Este projeto utiliza uma marca e produtos fictícios para fins educacionais.
